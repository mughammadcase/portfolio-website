# Developer Portfolio

## Overview

This project is my personal developer portfolio website, created to showcase my projects, skills, education, and professional experience. The goal was to build a clean, responsive, and professional portfolio that highlights my development journey while providing an easy way for potential employers, recruiters, and other developers to learn more about me.

The portfolio was built using HTML, CSS, and JavaScript, with a focus on simplicity, accessibility, responsiveness, and maintainable code.

---

## Live Demo

🔗 [View the live site on GitHub Pages](https://mughammadcase.github.io/portfolio-website/)

---

## Features

- Responsive design for desktop, tablet, and mobile devices
- Sticky navigation with smooth scrolling
- Mobile-friendly navigation menu
- Dark mode theme toggle with local storage persistence
- Featured project showcase with screenshots and project links
- Experience and education section
- Contact section with GitHub, LinkedIn, and email links
- Modern glassmorphism-inspired user interface
- Basic SEO implementation using meta tags

## Project Structure

```text
portfolio/
│
├── assets/
│   ├── images
│   └── favicon
│
├── css/
│   └── styles.css
│
├── js/
│   ├── main.js
│   ├── navigation.js
│   └── theme.js
│
└── index.html
```

### File Responsibilities

#### index.html

Contains the structure and content of the website, including:

- About section
- Featured projects
- Experience and education
- Contact information
- Footer

#### styles.css

Responsible for all styling, including:

- Layout and spacing
- Responsive design
- Glassmorphism effects
- Dark mode variables
- Mobile navigation styling
- Project cards and sections

#### theme.js

Handles theme management by:

- Loading the user's saved theme preference
- Applying the selected theme on page load
- Updating local storage when the theme changes

#### navigation.js

Handles mobile navigation functionality by:

- Opening and closing the mobile menu
- Automatically closing the menu when a navigation link is selected

#### main.js

Acts as the application's entry point and initializes all JavaScript modules when the page loads.

## Development Approach

When building this portfolio, I focused on creating a design that balances professionalism with modern UI practices. I wanted the site to remain lightweight while still providing an engaging experience through subtle animations, responsive layouts, and a theme toggle feature.

The project also served as an opportunity to reinforce core front-end development concepts such as semantic HTML, CSS architecture, modular JavaScript, accessibility considerations, and responsive design techniques.

## Running the Project Locally

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder

```bash
cd portfolio
```

3. Open the project

You can either:

- Open `index.html` directly in your browser, or
- Use the VS Code Live Server extension for local development
